# TypeScript-এ `keyof` কীওয়ার্ডের ব্যবহার (সহ উদাহরণ)?


TypeScript কোডকে আরো নিরাপদ এবং টাইপ-চেকড করতে সহায়তা করে। এর মধ্যে একটি গুরুত্বপূর্ণ ফিচার হচ্ছে `keyof`। এটি একটি অপারেটর যা একটি অবজেক্টের সমস্ত **key**-এর ইউনিয়ন টাইপ তৈরি করে।

`keyof` ব্যবহার করে TypeScript একটি অবজেক্ট টাইপের সকল প্রোপার্টির নামকে ইউনিয়ন টাইপ হিসেবে ফেরত দেয়।

### উদাহরণ:

```ts
interface User {
  name: string;
  age: number;
  email: string;
}

type UserKeys = keyof User;   -->
// ফলাফল:
// type UserKeys = "name" | "age" | "email"
```

### `keyof` এর সুবিধাসমূহ:

- **ডায়নামিকভাবে প্রোপার্টি অ্যাক্সেস** করতে সক্ষম হওয়া।
- **জেনেরিক ফাংশনে টাইপ সুরক্ষা** বাড়ানো।
- **অবজেক্টের key গুলো**কে টাইপ হিসেবে ব্যবহার করা।




# TypeScript-এ `enum` কী এবং এটি কিভাবে কাজ করে?

 

TypeScript-এ enum (enumeration) হলো একটি শক্তিশালী ফিচার যা বিভিন্ন নামকৃত কনস্ট্যান্টের একটি সেট তৈরি করে। এটি কোডের readability, maintainability এবং error-reduction এর দিক থেকে গুরুত্বপূর্ণ ভূমিকা পালন করে, বিশেষত যখন  কোডে নির্দিষ্ট কিছু স্থির মান ব্যবহৃত হয়।

### `enum` এর সুবিধাসমূহ: 

- কোডের পাঠযোগ্যতা বৃদ্ধি পায়।
- একই সেটের মান বার বার ব্যবহার করা সরল হয়।
- টাইপ-সেফ কনস্ট্যান্ট সেট তৈরি করা যায়, যা ভুল বানানের সম্ভাবনা কমিয়ে দেয়।

###  Numeric Enum
TypeScript-এ Numeric enum এর মাধ্যমে  সংখ্যা দিয়ে enum তৈরি করা যায় । এটি ডিফল্টরূপে 0 থেকে শুরু করে এবং পরবর্তী মানগুলি স্বয়ংক্রিয়ভাবে একে অপরের পরবর্তী সংখ্যা গ্রহণ করে।  উদাহরণ:

```typescript
enum Direction {
    Up,
    Down,
    Left,
    Right
}

// ব্যবহার
let move: Direction = Direction.Up;
console.log(move); 
```

এছাড়া, যখন নির্দিষ্ট মানকে একসঙ্গে সেট করার সময় তা কাস্টমাইজ করা যায়। যেমন :

```typescript
enum Direction {
    Up = 1,
    Down,
    Left,
    Right
}

// ফলস্বরূপ, Down হবে 2, Left হবে 3, এবং Right হবে 4।
```
